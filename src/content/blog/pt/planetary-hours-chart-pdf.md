# Tabela de Horas Planetárias: Guia de Referência Imprimível Gratuito

Se você trabalha com horas planetárias com frequência, uma boa tabela de referência economiza tempo todos os dias. É o tipo de material que vale deixar fixado no escritório, salvo no celular ou no seu caderno de prática.

**Download rápido:** [Abrir versão imprimível](/downloads/planetary-hours-chart.html) → Imprimir → Salvar como PDF.

Para horários exatos por cidade e data, use a [Calculadora de Horas Planetárias](/pt).

## O que essa tabela entrega

Uma tabela de horas planetárias mostra duas bases essenciais:

1. **A sequência fixa dos planetas** (Ordem Caldeia).  
2. **O planeta regente de cada dia da semana**.

Com isso, você planeja melhor quando comunicar, negociar, lançar, estudar, descansar ou estruturar tarefas profundas.

## Ordem Caldeia (a engrenagem principal)

A sequência tradicional é:

**Saturno → Júpiter → Marte → Sol → Vênus → Mercúrio → Lua**

Essa roda se repete hora após hora. O que muda de um dia para outro é apenas o planeta que abre a primeira hora após o nascer do sol.

Quer base completa? Leia [Horas Planetárias Explicadas](/pt/blog/what-are-planetary-hours).

Contexto extra em [Dias Planetários da Semana](/pt/blog/planetary-days-of-the-week).

## Referência rápida de uso (modo fã editor)

- **Sol:** liderança, presença, visibilidade.  
- **Lua:** cuidado emocional, casa, vínculo.  
- **Mercúrio:** comunicação, estudo, contratos.  
- **Vênus:** relacionamentos, estética, colaboração.  
- **Marte:** execução, coragem, decisão.  
- **Júpiter:** expansão, oportunidade, visão.  
- **Saturno:** estrutura, disciplina, foco de longo prazo.

## Como usar tabela + calculadora

1. Defina a atividade (ex.: negociação).  
2. Escolha o planeta ideal (Mercúrio/Júpiter, conforme objetivo).  
3. Abra a [Calculadora](/pt) para obter o horário **exato** na sua cidade.  
4. Execute nessa janela.

A tabela oferece estratégia; a calculadora entrega precisão tática.

## Como salvar em PDF

### No desktop

1. Pressione **Ctrl+P** (ou **Cmd+P** no Mac).  
2. Selecione **Salvar como PDF**.  
3. Se quiser, use orientação horizontal para tabelas largas.  
4. Salve.

### No celular

1. Abra compartilhar/menu do navegador.  
2. Toque em **Imprimir**.  
3. Escolha **Salvar como PDF**.  
4. Conclua.
